# <img src="https://raw.githubusercontent.com/baicie/orix/main/packaging/appimage/orix.png" width="32" height="32" style="vertical-align: middle;" /> @orix/orix-darwin-arm64

Native binary for [@orix/orix](https://github.com/baicie/orix) on macOS ARM64.

This package is automatically selected by the main `@orix/orix` package based on your OS and CPU architecture.

## Platform

- **OS:** darwin
- **CPU:** arm64

## Installation

This package is installed automatically when you install `@orix/orix`:

```bash
npm install -g @orix/orix
# or
pnpm add -g @orix/orix
```

## Description

High-performance package manager written in Rust, compatible with pnpm.

See the main repository for details: [https://github.com/baicie/orix](https://github.com/baicie/orix)
