# @orix-pnpm/orix-win32-x64-msvc

Native binary for [@orix-pnpm/orix](https://github.com/baicie/orix) on Windows x64 MSVC.

This package is automatically selected by the main `@orix-pnpm/orix` package based on your OS and CPU architecture.

## Platform

- OS: win32
- CPU: x64

## Installation

This package is installed automatically when you install `@orix-pnpm/orix`:

```bash
npm install -g @orix-pnpm/orix
# or
pnpm add -g @orix-pnpm/orix
```

## Description

High-performance package manager written in Rust, compatible with pnpm.

See the main repository for details: https://github.com/baicie/orix
