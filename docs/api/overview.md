# API Reference Overview

This section documents the public API of each crate in the workspace.

## Crates

| Crate | Description |
|-------|-------------|
| [CLI](./cli) | Command-line interface entry point |
| [Core](./core) | Core business logic and domain types |
| [Config](./config) | Configuration loading and validation |
| [Utils](./utils) | Shared utility functions and helpers |
| [Macros](./macros) | Procedural macro definitions |

## Version

See `Cargo.toml` for the current version. All crates share the same version number.

## MSRV

The minimum supported Rust version (MSRV) is **1.80**.
